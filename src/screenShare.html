Why Screen Sharing Fails?

  1. You've not used 'chromeMediaSource' or 'mediaSource' constraint: 
                        // for chrome
                    mandatory: {chromeMediaSource: 'screen'}
                    // or desktop-Capturing
                    mandatory: {chromeMediaSource: 'desktop'}

                    // for Firefox
                    video: {
                        mediaSource: 'window' || 'screen'
                    }
